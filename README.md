# Portfólio — Henrique Poltronieri

> Site de portfólio pessoal desenvolvido com **HTML, CSS e JavaScript**, com design minimalista e elegante.

---

### 🚀 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível
- **CSS3** — Variáveis CSS, Flexbox, Grid, animações e responsividade
- **JavaScript** — Sem frameworks ou bibliotecas externas
- **Google Fonts** — Syne (títulos) + DM Sans (corpo)

### 📌 Seções

| Seções | Descrição |
|-------|-----------|
| **Hero** | Apresentação com nome, título, bio, botões CV e redes sociais |
| **Skills** | Grid de tecnologias: Mobile, Back-end, Banco de Dados e Ferramentas |
| **Projetos** | Cards filtráveis por categoria |
| **Cursos** | Cotemig e Alura |
| **Contato** | Informações de contato + formulário de envio de mensagem |

---

### ⚙️ Funcionalidades

- **Barra de progresso de scroll** — indicador visual no topo da página
- **Navbar dinâmica** — fica opaca com backdrop-blur ao rolar
- **Menu responsivo** — hamburger menu para dispositivos móveis
- **Animações de entrada** — cards aparecem suavemente via `IntersectionObserver`
- **Filtro de projetos** — filtra os cards por tag com animação
- **Destaque de seção ativa** — link do menu muda de cor conforme o scroll
- **Cursor glow** — efeito de brilho sutil que acompanha o cursor (desktop)
- **Anel animado no avatar** — anéis giratórios com `CSS animation`
- **Badges flutuantes** — tags tecnológicas com animação de float
- **Formulário funcional** — feedback visual de envio de mensagem

---

### 📱 Responsividade

O layout é totalmente responsivo com breakpoints em:

- **≤ 1024px** — Grid de projetos passa para 2 colunas; cursos empilham verticalmente
- **≤ 860px** — Hero empilha verticalmente; avatar redimensionado
- **≤ 640px** — Menu colapsável (hamburger); projetos em coluna única
